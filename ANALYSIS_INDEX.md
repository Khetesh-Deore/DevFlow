## Quick Reference

### Technology Stacks

**DevFlow**
- Frontend: React 18 + TypeScript + Vite + Tailwind
- Backend: Express + Node.js + MongoDB + Bull + Redis
- Execution: Docker Sandbox
- Real-time: Socket.io

**FireCode**
- Frontend: React 18 + TypeScript + CRA + Tailwind
- Backend: Express + TypeScript + MongoDB + Mongoose
- Execution: None
- Real-time: None

**LeetCode Clone**
- Frontend: React 18 + TypeScript + Vite + Tailwind
- Backend: Firebase Cloud Functions
- Database: Firestore
- Real-time: Firestore Listeners

**Codeforces Clone**
- Frontend: Vue.js 2 + JavaScript + Vue CLI
- Backend: Spring Boot + Java
- Database: MySQL
- Real-time: None
